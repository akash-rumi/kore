## About This Project

KoreSearch is a Learning Management System and course marketplace platform built with Laravel 10. The project includes a public Home page, Course listings, a Cart, a bKash-based Checkout flow, and an Admin Dashboard.

This project is provided as a developer assessment task. You are expected to set it up locally, explore the application, identify issues, fix them, and submit your work.
